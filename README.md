# 進撃のこうかとん

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1
* 必要なものがあれば追記してください（非推奨）

## ゲームの概要
* 片柳研究所から出撃するこうかとんたちをタワーで迎撃し、落としたチキンで防衛を強化していくタワーディフェンスゲーム。
* 参考URL：[サイトタイトル](https://www.hoge.com/)

## ゲームの遊び方
* 画面下のタワー購入ボタンを押して、こうかとんが通る道の横にタワーを設置する。
* タワーの購入にはチキンが必要。
* チキンはこうかとんを撃退すると獲得できる。

## ゲームの実装
### 共通基本機能
* 背景画像とタワー、敵キャラクターの描画
* タワーの設置、攻撃
* こうかとんの出撃、進軍、退散

### 分担追加機能
* リスタート可能なゲームオーバー（担当：A）：ゲームオーバーの処理、リスタートの処理を行うクラス
* タワーの強化（担当：B）：タワーをチキンを消費して強化できる機能
* トラップの設置（担当：C）：こうかとんの進行を妨害するトラップを設置できるようにする機能
* エリートこうかとん（担当：D）：ときどき強いこうかとんが出現するようになる機能
* フィーバータイム（担当：E）：一定の条件を達成するとフィーバータイムが発生してこうかとんの出撃数とタワーの攻撃速度が上がり、背景が賑やかになる機能

### メモ
* クラス内の変数は，すべて，「get_変数名」という名前のメソッドを介してアクセスするように設計してある
* すべてのクラスに関係する関数は，クラスの外で定義してある
