# 進撃のこうかとん

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1
* 必要なものがあれば追記してください（非推奨）

## ゲームの概要
* 片柳研究所から出撃するこうかとんたちをタワーで迎撃し、落としたチキンで防衛を強化していくタワーディフェンスゲーム。

## ゲームの遊び方
* 画面下のタワー購入ボタンを押して、こうかとんが通る道の横にタワーを設置する。
* チキンはこうかとんを撃退すると獲得できる。

## ゲームの実装
### 共通基本機能
* 背景画像とタワー、敵キャラクターの描画
* タワーの攻撃
* こうかとんの出撃、進軍、退散

### 分担追加機能
* リスタート可能なゲームオーバー（担当：高原）：ゲームオーバーの処理、リスタートの処理を行うクラス
    * 詳細：ゲームオーバーになったときに画面表示をして、qを押すと閉じてｒを押すと再び始められるようにした
* 敵の強さが時間経過ごとに耐久力と、速度が上昇するようにした
* タワーの設置と強化（担当：表野）：タワーをチキンを消費して設置、強化できる機能
    * 詳細：マスをクリックすることで、チキンを100消費し、新しいタワーを設置することができる
        * タワーがあるマスを再度クリックすると、タワーを強化することができる
        * 射出までのクールダウンが減少し、検出レンジが増加する
        * レベルは１〜５まで
        * 基本コストは50だが、レベルに基本コストを掛け合わせてコストが計算される
* トラップの設置（担当：箕輪）：こうかとんの進行を妨害するトラップを設置できるようにする機能
    * 詳細：敵の軌道上で右クリックをすると道にオレンジ色のトラップが出現し、敵に20ダメージを与える。敵に1回当たったら消える。使用時に50チキン消費する
* エリートこうかとん（担当：竹庵）：ときどき強いこうかとんが出現するようになる機能
* フィーバータイム（担当：中馬）：一定の条件を達成するとフィーバータイムが発生してこうかとんの出撃数とタワーの攻撃速度が上がり、背景が賑やかになる機能
    * 詳細：こうかとんの撃退数が一定に達するとFキーでフィーバーモード発生。背景が金色になり、こうかとんの出撃数が増加。タワーの攻撃速度も上がり、スコアを一気に稼げるようになる。

### メモ
* クラス内の変数は，すべて，「get_変数名」という名前のメソッドを介してアクセスするように設計してある
* すべてのクラスに関係する関数は，クラスの外で定義してある

### エリートこうかとん
* 一定確率でエリートこうかとんが発生する。
* エリートこうかとんは速度、体力が高く、撃破時の報酬が多くなっている。
